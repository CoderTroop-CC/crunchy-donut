# crunchy-donut
A note taking app with collaborative comments.

## Development server

Run `ng serve` and `node server.js` for a dev server. Navigate to `http://localhost:4200/`. 

## Software Stack

* MongoDB (mLab)
* Express
* Angular
* Node
